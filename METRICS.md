# Product Metrics (KPIs)

1. **Conversion Rate:** Percentage of users who download the PDF report after calculating.
2. **Average Savings Identified:** The mean dollar amount of annual savings found per audit.
3. **User Retention:** Number of returning users checking updated tool prices.
4. **Data Accuracy:** Monthly verification of AI tool prices in `pricing.ts`.

## North Star Metric
- **Total Annual Savings Identified:** The primary goal is to show the massive value Credex provides to the startup ecosystem.

## Pivot Trigger
- If the **Average Savings Identified** drops below **$200/year**, it indicates that startups are already optimizing their stacks or market prices have crashed. In this case, we would pivot the tool toward **AI Security Compliance** rather than cost saving.
